## Deploy on Vercel

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Vercel auto-detects Next.js and builds automatically
4. Done!

**Build Command:** `npm run build`  
**Output Directory:** .next (auto-configured)

## Project Structure
