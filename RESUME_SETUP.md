## Quick Setup

1. **Save your resume as PDF**
   - File name: `resume.pdf`
   - Location: `public/resume.pdf`

2. **How to create/convert your resume to PDF:**
   - **Option A (Recommended):** Save from Google Docs
     - Open your resume in Google Docs
     - File → Download → PDF Document
     - Save as `resume.pdf`
   
   - **Option B:** From Word/LibreOffice
     - Open your resume in Word or LibreOffice
     - File → Export as PDF
     - Save as `resume.pdf`
   
   - **Option C:** Online converter
     - Upload your resume file to an online PDF converter
     - Download as PDF
     - Save as `resume.pdf`

3. **Place the file in your project:**
   - Save `resume.pdf` to the `public/` folder
   - Full path: `c:\Developer Moises\ai-tool-nextjs-main\public\resume.pdf`

## Verification
