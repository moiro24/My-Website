# Resume File

Place your resume PDF file here as:
- `public/resume.pdf`

The resume will be available for download via:
- Header "Resume" button → downloads as `Moises_Rosas_Resume.pdf`
- Direct link: `/resume.pdf`

**Steps:**
1. Create your resume in PDF format (Google Docs, Word, Canva, etc.)
2. Export/save as PDF
3. Place file in `public/resume.pdf`
4. The download link will automatically work

This file is referenced in:
- `src/components/Header/index.tsx` - Resume download button
